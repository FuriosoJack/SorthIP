# SorthIP
Ordena lista de direcciones ip y subredes | Sort list ip addresses and subnets

## Installation 

```bash
$ php composer require furiosojack/sorthip=^0.0
```

OR 

add to your `composer.json`

```json
{
    "require": {
        "furiosojack/sorthip": "^0.0"
    }
}
```
